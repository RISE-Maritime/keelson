Keelson mediamtx — WHEP debug client

Minimal browser client for the mediamtx connector's WHEPProxy interface. It performs the WebRTC/WHEP handshake by sending a Zenoh GET query (a WHEPRequest) to the connector's whep_signal queryable and applying the returned WHEPResponse SDP. It talks to Zenoh over the zenoh-ts remote-api WebSocket — see the README for the required router plugin.
